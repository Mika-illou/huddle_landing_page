
# Description du Projet

Huddle est une page de destination moderne et responsive qui présente une plateforme de communauté virtuelle. Le projet vise à démontrer une conception web attrayante avec un accent sur la création de communautés engagées entre les créateurs de contenu et leur audience.

# Fonctionnalités Clées

Design Responsive - S'adapte parfaitement à tous les appareils
Interface Moderne - Design épuré avec des éléments visuels attractifs
Sections Informatives - Présentation détaillée des fonctionnalités principales
Call-to-Action - Boutons incitatifs pour l'engagement utilisateur
Footer Complet - Informations de contact et liens utiles

# Technologies Utilisées

HTML: Structure sémantique du contenu
CSS: Styles avancés et mise en page responsive
Media Queries: Adaptation aux différentes tailles d'écran

# Démarche du Projet

Le projet suit une approche mobile-first avec des points de rupture stratégiques :
1440px et 1024px: Adaptation pour les grands écrans et ordinateurs de bureau
768px:  Optimisation pour les tablettes
425px, 375px, 320px: Adaptation mobile avancée

# Adaptation Mobile

Le site est entièrement responsive avec des adaptations spécifiques :
Réorganisation flexible des sections selon la taille d'écran
Ajustement des typographies pour une lisibilité optimale
Redimensionnement intelligent des images et boutons
Navigation simplifiée sur les petits écrans
Optimisation des espacements et marges

# Caractéristiques de Design

Palette de couleurs harmonieuse avec accent sur le rose (#FE7DD2) et violet (hsl(322, 100%, 66%))
Utilisation d'ombres portées pour créer de la profondeur
Cartes avec coins arrondis pour un look moderne
